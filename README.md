# Buenas Viagens!

Site de turismo desenvolvido como avaliação para a vaga de 
Desenvolvedor Front-End Jr na BGTech.

## Sobre o projeto
O layout foi desenvolvido com base no material de design 
fornecido pelo professor, reproduzindo as seções, cores, 
tipografia e componentes apresentados no arquivo.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Seções desenvolvidas
- Navbar com links de navegação
- Hero com imagem de fundo e barra de busca
- Seção de Próximos Passeios & Destinos
- Grid de cards de destinos com preços
- Footer com redes sociais

## Design de referência
O projeto foi baseado no arquivo de design disponibilizado 
pelo professor.
